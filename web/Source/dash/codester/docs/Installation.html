Installation of ApPHP WebsiteCleaner


A new installation of ApPHP WebsiteCleaner is a very straight forward process:

Step 1. Uncompressing downloaded file.

Uncompress the ApPHP WebsiteCleaner version x.x.x script archive. The archive will create a directory called "PHPWSC_xxx".


Step 2. Uploading files.

Upload content of sub-directory website-cleaner/ (all files and sub-directories it includes) to your document root (public_html, www, httpdocs etc.) or your "website-cleaner directory" using FTP.

Pay attention: DON'T use the capital letters in the name of the folder (for Linux users).

For example: 
public_html/
or 
public_html/{website-cleaner directory}/


Step 3. Running example file.

Now you can run the example files. To do this, open a browser and type in Address Bar following: 
http://localhost/{website-cleaner directory}/index.php
or 
http://{www.mydomain.com}/{website-cleaner directory}/index.php


Congratulations, you now have ApPHP WebsiteCleaner installed!
